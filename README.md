# ManhattanBoroughProject

Manhattan Culture & Commerce Explorer App
Global Connections: finance, arts, tourism, international communities

Frontend: HTML, CSS, JavaScript
Backend: Java Spring Boot

Classes 
1. Map
2. Save 
    - user needs to be able to add a point
3. Button for text review
    - Save button to save text review
4. Button to open
5. Review
    - can like other reviews
6. click a point on the map to see details



Map will be on homepage at the top, and when a bullet point is chosen, a modem pops up to show reviews on the side. Each bullet point will have buttons for leaving a review, opening and updating and deleting.
